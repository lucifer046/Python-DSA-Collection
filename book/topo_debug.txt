on a **shirt before undershirt**. Some tasks MUST happen before others!\n\n**Topological Sort** finds a **legal 
order** to complete all tasks so that every task starts AFTER its prerequisites are done.\n\n> **Simple Definition:** 
Given tasks with dependencies (Task A must finish before Task B can start), Topological Sort finds a valid order to
